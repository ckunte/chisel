About

{% markdown %} I am an offshore structures engineer, and this little website is my sandbox. Here I love to code, experiment, write. Currently, I live in Perth. __ckunte.net__ exists mainly to be a pointer to an __MX__ record. I put the idling __A__ record to use, mostly as an afterthought, which turned-on the lights here. Built with [Chisel][c1], this site respects web standards, produces valid [HTML][v1], [CSS][v2], [RSS][v3] markups, and is phone-friendly. Handy keyboard shortcuts are available as below. On MacOS: Ctrl + Opt + Key.
